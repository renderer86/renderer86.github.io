## About

실시간 렌더링과 게임 엔진 구조를 공부하고 정리하는 블로그입니다.

과거 Nexon, NCSoft에서 그래픽스 프로그래머로 있었으며,
현재 AButton에서 엔진 및 그래픽스 역할을 하고 있습니다.

이 블로그는 단순히 "무엇을 쓰면 된다"보다, 왜 이런 구조가 필요했고 엔진 안에서는 어떤 순서로 동작하는지에 관심을 둡니다. 공식 문서, 발표 자료, 논문, 엔진 코드를 함께 보면서 기술의 목적과 구현 흐름을 정리합니다.

## Topics

- Real-time Rendering
- Unreal Engine Renderer
- Graphics Programming
- Ray Tracing / Distance Field / Lumen
- Nanite / Virtual Shadow Map / MegaLights
- Engine Architecture
- AI Tools for Development

## Writing Style

글은 개인적인 학습 기록이면서, 나중에 다시 봐도 구현 흐름을 빠르게 복원할 수 있도록 정리한 기술 노트입니다.

가능하면 개념 설명에서 끝내지 않고, 실제 엔진 코드나 렌더링 파이프라인 안에서 그 개념이 어디에 놓이는지까지 함께 봅니다.
